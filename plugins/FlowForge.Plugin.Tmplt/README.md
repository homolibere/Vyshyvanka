# FlowForge.Plugin.Template

A starter template for building [FlowForge](https://github.com/your-org/FlowForge) plugins.

## Nodes

| Node | Type | Description |
|------|------|-------------|
| Sample Action | `sample-action` | Echoes input with a configurable greeting |

## Quick start

```bash
dotnet build
dotnet pack src/FlowForge.Plugin.Template.csproj -c Release
```

See [docs/getting-started.md](docs/getting-started.md) for full setup instructions.
