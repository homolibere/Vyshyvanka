# FlowForge Product Overview

FlowForge is a workflow automation platform built on .NET 10, inspired by n8n. It enables users to create, manage, and execute automated workflows through a visual node-based interface.

## Core Capabilities

- Visual drag-and-drop workflow designer (Blazor WebAssembly)
- Node-based workflow execution engine with trigger, action, and logic nodes
- Expression evaluation for data transformation between nodes
- Secure credential management with AES-256 encryption
- REST API and webhook support for external integrations
- Plugin system for custom node extensions
- Role-based access control (Admin, Editor, Viewer)
- Execution history and debugging tools

## Key Concepts

- **Workflow**: Collection of connected nodes defining an automated process
- **Node**: Building block performing a specific action (Trigger, Action, Logic)
- **Connection**: Link between nodes defining data flow
- **Execution**: Single run of a workflow with tracked state
- **Credential**: Encrypted authentication data for external services
